# Data

This folder is intentionally empty. PDF documents are not included 
in this repository and must be downloaded directly from the 
World Bank Open Knowledge Repository.

## How to Download Documents

### Step 1 — Go to the World Bank Open Knowledge Repository
Visit: https://openknowledge.worldbank.org

### Step 2 — Search for Implementation Completion Reports
In the search bar, search for terms like:
- "Implementation Completion Report Nepal education"
- "Implementation Completion Report Bangladesh education"
- "Implementation Completion Report Nepal health"

### Step 3 — Filter by Document Type
On the left sidebar, filter by document type and select 
"Implementation Completion Report" or "ICR" to ensure 
you are downloading structured outcome documents.

### Step 4 — Download PDFs
Click on a result, then click the Download or PDF button 
on the document page. Save files to this data/ folder.
